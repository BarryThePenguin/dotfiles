# Dotfiles

OS X configuration files. There's no place like ~/.

## Installation

```shell
$ mkdir -p ~/src/btp
$ git clone https://github.com/BarryThePenguin/dotfiles.git ~/src/btp/dotfiles
$ cd ~/src/btp/dotfiles
$ ./install
```
